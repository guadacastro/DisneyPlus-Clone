### How To Install

#### Clone the repository
```console
git clone https://github.com/guadacastro/DisneyPlus-Clone.git
```
#### Enter to the repository folder
 ```console
cd DisneyPlus-Clone
```
#### Install all dependencies
 ```console
npm install
``` 
#### Start the development server
 ```console
npm start
```
